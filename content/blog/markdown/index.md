Markdown — это простой язык разметки текста, который позволяет быстро оформлять заголовки, списки, ссылки и другие элементы, не отвлекаясь на сложные теги. Его главная идея: текст в Markdown остаётся удобочитаемым даже в исходном виде.

## Основные конструкции

- # Заголовок — создаёт заголовок (чем больше #, тем меньше уровень).
- **жирный** или *курсив* — выделение текста.
- - пункт или 1. пункт — ненумерованные и нумерованные списки.
- [текст](ссылка) — вставка гиперссылки.
- ![alt](image.jpg) — вставка изображения.

## Зачем это мне?

Я использую Markdown для написания всех постов на этом сайте. Hugo автоматически превращает Markdown-файлы в HTML-страницы. Это удобно, быстро и не требует знания веб-вёрстки. Кроме того, Markdown поддерживается на GitHub, в редакторах заметок (Obsidian, Notion) и многих других инструментах.
